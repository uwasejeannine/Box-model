CSS BOX MODEL

What is box model

Box model refer to how htlm element are modeled in browser engines and how
dimesion of those html element are derived from css properties

They are three main properties of box model.

content- The content of the box, where text and images.

Padding-Clear an area around content. The padding is transparent.

Border- A border tht goes round the padding and content.

Margin- clear area outside the border.The margin is transparent.

Total width=width+left padding+right padding+left margin+right margin

400=(20*4 +20*4 px +2*4)
